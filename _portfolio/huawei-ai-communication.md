## Overview

During my time at Huawei Technologies Co., Ltd. (2021-2024), I worked on several projects that integrated artificial intelligence technologies with communication systems. These projects focused on improving communication efficiency, network optimization, and intelligent system design.

## Key Projects

### Intelligent Network Optimization
- Developed AI-based algorithms for network resource allocation and optimization
- Implemented machine learning models for traffic prediction and management
- Improved network efficiency and reduced latency through intelligent routing

### AI-Enhanced Communication Protocols
- Researched and developed communication protocols enhanced with AI capabilities
- Designed adaptive systems that learn from network conditions
- Optimized protocol parameters using reinforcement learning techniques

## Technologies Used

- Deep Learning (PyTorch, TensorFlow)
- Reinforcement Learning
- Network Optimization Algorithms
- Communication Protocol Design
- Distributed Systems

## Impact

These projects contributed to improving communication system performance and efficiency, demonstrating the potential of AI-communication integration in practical industrial applications.
